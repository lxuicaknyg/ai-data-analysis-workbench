## 注意事项

1. **可逆操作**: 所有隐藏改动均采用 HTML 注释方式，未删除任何代码，可随时恢复
2. **刷新生效**: 修改后需刷新浏览器页面才能看到效果
3. **统一风格**: 注释格式统一为 `<!-- 描述 - 日期 -->`，便于查找
4. **样式调整**: 弹窗尺寸和对齐样式可直接修改数值，无需取消注释

## 取消注释示例

### 示例 1：恢复"立即使用"按钮

**修改前（被注释的代码）**：
```vue
<!-- 隐藏"立即使用"按钮 - 2026-05-27 -->
<!-- <NTooltip
  trigger="hover"
  :overlay-style="tooltipOverlayStyle"
  :content-style="tooltipContentStyle"
  :theme-overrides="tooltipThemeOverrides"
  placement="bottom"
  to="body"
  :flip="true"
>
  <template #trigger>
    <NButton size="small" quaternary circle @click.stop="$emit('use', favorite)">
      <template #icon><NIcon><PlayerPlay /></NIcon></template>
    </NButton>
  </template>
  {{ t('favorites.manager.card.useNow') }}
</NTooltip> -->
```

**修改后（取消注释）**：
```vue
<!-- 隐藏"立即使用"按钮 - 2026-05-27 -->
<NTooltip
  trigger="hover"
  :overlay-style="tooltipOverlayStyle"
  :content-style="tooltipContentStyle"
  :theme-overrides="tooltipThemeOverrides"
  placement="bottom"
  to="body"
  :flip="true"

```

**操作步骤**：
1. 找到以 `<!-- 隐藏"立即使用"按钮 - 2026-05-27 -->` 开头的部分
2. 删除代码块两端的 `<!--` 和 `-->` 注释标记
3. 保留说明注释（第一行）以便日后追溯

### 示例 3：调整分页导航对齐

**当前样式**：
```css
.content-wrapper {
  width: calc(100% + 64px);
  margin-left: -32px;
  margin-right: -32px;
}
```

**如需更大延伸**：
```css
.content-wrapper {
  width: calc(100% + 80px);  /* 增大扩展值 */
  margin-left: -40px;        /* 增大负边距 */
  margin-right: -40px;
}
```

**如需更小延伸**：
```css
.content-wrapper {
  width: calc(100% + 48px);  /* 减小扩展值 */
  margin-left: -24px;        /* 减小负边距 */
  margin-right: -24px;
}
```
